# node-cleanup
Helps Windows web developers remove node_modules folder.

### Current Version: 0.2.3

## Getting Started
1. Place ```cleanup.js``` in your project root.
2. ```npm install --save-dev colors del```

## Default Usage
```node cleanup.js```

## Alternative Usage
```node cleanup.js /path/to/directory```
