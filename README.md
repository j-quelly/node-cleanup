# node-cleanup
Helps Windows web developers remove node_modules folder.

### Current Version: 1.5.15

## Getting Started
1. $ npm install -g remove-node-modules

## Default Usage
$ remove-node-modules

## Alternative Usage
$ remove-node-modules /path/to/directory
